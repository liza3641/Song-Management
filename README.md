# SongManagement
---


## **프로젝트 개요**

### **설명**

> MySQL을 활용한 노래 관리 프로그램


### **사용기술**

> C#, Winform, MySQL, Github

<br>

---

## **결과물**

### **메인 화면**

> 실시간으로 DB의 값을 가져와 화면에 출력<br>
> 조건을 입력하고 검색하면 조건을 만족하는 값들만 출력 가능<br>
> 값이 들어 있는 Cell을 클릭하면 정보수정이 가능<br>
